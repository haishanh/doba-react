To run in your local environment.

```sh
$ git clone https://github.com/haishanh/doba-react.git
$ cd doba-react
$ yarn
$ yarn start
```

Then open `http://localhost:3000` in your browser.

To build for production usage:

```sh
$ yarn build
```
